portfolio website for cs50 week 8

## files needed

- index.html
- about.html
- skills.html
- contact.html
- styles.css
- Media folder with images

## how to use

1. put all files in one folder
2. create Media folder
3. add your 13 images
4. open index.html in browser

done.

## 7 colour themes

switch between themes:
- desktop: click Theme button (top right)
- mobile: tap coloured squares in menu

themes save automatically

1. lavender aroma
2. serene skies (default)
3. outdoor yoga
4. alluring apothecary
5. wellness spa
6. vibrant bowl
7. starry blue

## common issues

**images not showing** - check file names match exactly

**theme not saving** - enable cookies/localStorage

**mobile menu stuck** - refresh page

## adding more pages

1. copy any html file
2. change middle content

4. done
