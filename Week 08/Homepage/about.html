Background & Approach

I'm a developer who works primarily in Python, C, SQL, and Swift for iOS development. My background combines strong programming fundamentals with practical, real-world application development.

Unlike many developers who rely entirely on frameworks and libraries, I've spent time understanding how things work beneath the surface—memory management, algorithm efficiency, data structures. This means I can debug problems others might struggle with and build systems that don't fall apart under pressure.

I've worked on everything from small business iOS apps to larger web platforms, always with the same goal: create something that works properly and makes life easier for whoever uses it.

Education & Foundations

Through CS50x and CS50's Introduction to Artificial Intelligence with Python, I've developed solid foundations in:

Algorithms and problem-solving

Understanding how to approach complex problems systematically rather than guessing and hoping.

Data structures and efficiency

Knowing when to use which structure and why it matters for performance.

Machine learning and AI concepts

Not just using AI tools, but understanding how they function and when they're appropriate.

These courses emphasise thinking before coding, which prevents the kind of fragile, patch-covered systems that plague many projects. You design it properly first, then you build it properly.

Development Philosophy

Good software should be simple to use, logical in structure, and built with clear purpose.

I don't believe in overengineering. If a straightforward solution exists, that's the one to use. Complexity should only appear when the problem genuinely requires it.

When I write code, I write it as if someone else will need to maintain it—because they probably will. Clear naming, logical organisation, and proper documentation aren't optional extras; they're fundamental requirements.

Most importantly, I build with real users in mind. An application that solves a problem but frustrates everyone who uses it hasn't actually solved anything.
