- **SQL & SQLite** — relational databases, queries, schema design
- **HTML, CSS, JavaScript** — frontend development and Bootstrap

## Highlights

**Week 9 — Flask**
Two full web applications built with Flask, SQLite, and Python:

- **Birthday Tracker** — a full CRUD birthday management app with inline editing, server-side validation, and a fully custom "Serene Skies" CSS theme in warm peach and slate tones
- **C$50 Finance** — a full-stack stock trading web app with real-time stock price lookups, user authentication, portfolio management, transaction history, and personal features including password change, cash deposit, and withdrawal
